# Money Manager - Frontend

An advanced personal and business finance management system designed to provide clarity and organization to your financial life. Built with React and Tailwind CSS, this frontend provides an intuitive interface for tracking income, expenses, and analyzing financial trends.

## üöÄ Overview

Money Manager bridges the gap between simple expense trackers and complex accounting software. It offers a professional-grade dashboard with real-time financial insights, enabling users to make informed decisions with confidence.

### Key Features

- **Dual-Division Tracking:** Specialized views for separating personal and office finances.
- **Dynamic Dashboard:** Real-time summary cards for Income, Expenses, Net Balance, and Transaction counts.
- **Interactive Visualizations:** Deep financial insights through Line, Pie, and Bar charts powered by Recharts.
- **Smart Transaction Management:** Quick add/edit workflow with intelligent categorization.
- **Security-Focused Editing:** 12-hour window protection for transaction edits to ensure data integrity.
- **Advanced Filtering:** Multi-parameter search and reporting for deep dives into spending history.
- **Responsive Professional UI:** Sleek, modern design that works seamlessly across desktop, tablet, and mobile devices.

## üì• Installation Instructions

Follow these steps to set up the project locally:

1.  **Clone the Repository:**

    ```bash
    git clone https://github.com/your-username/money-manager.git
    cd money-manager/money-manager-frontend
    ```

2.  **Install Dependencies:**

    ```bash
    npm install
    ```

3.  **Environment Setup:**
    Create a `.env` file in the root directory (or update the existing one):
    ```env
    VITE_API_URL=http://localhost:5000/api
    VITE_APP_NAME=Money Manager
    ```
